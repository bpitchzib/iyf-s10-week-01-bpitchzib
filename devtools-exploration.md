# DevTools Exploration

## What I Learned

**Elements Tab**: View and edit HTML/CSS in real-time
**Console Tab**: JavaScript debugging and errors
**Network Tab**: See all files loading (HTML, images, CSS)

## Practical Exercise Completed

1. Opened DevTools (F12 on Chrome)
2. Inspected heading elements → saw computed styles
3. Edited text temporarily → page updated live

## Key Takeaway
DevTools helps debug layout issues without changing source files.
